# Collection - List

* List 특징 : 순서가 있고, 데이터 중복 가능
 
- ArrayList 
    - 배열 기반 구조
    - 데이터 조회가 빠르다
    - 데이터 추가/삭제 시 내부 로직이 복잡하여 오래 걸린다. 

- LinkedList
    - 리스트 기반 구조
    - 데이터 추가/삭제가 빠르다. 
    - 데이터 조회가 느리다. 

- method 
     - 데이터 추가 : add(데이터)
     - 데이터 삭제 : remove(index)
     - 데이터 조회 : get(index)
     - 데이터 수정 : set(index, data)
     - 저장된 데이터 크기 : size()
